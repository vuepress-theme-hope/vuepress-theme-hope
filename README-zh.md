## 详情

V2 基于 [**VuePress2**](https://vuejs.press/zh/)，通过 [**Vite7**](https://cn.vite.dev/) / [**Webpack5**](https://webpack.docschina.org/) 和 [**Vue3**](https://cn.vuejs.org/) 提供强大功能。

V2 完全重写，包括：

- 所有组件都用 **Vue3 中的 Composition API** 重写
- 所有样式都迁移到 **Sass**

V2 也有更好的性能 🚀：

- DevServer 冷启动时间减少了 **70%**
- 构建内存使用量减少了 **65%**
- 构建时间减少了 **75%**
- 输出尺寸减少了 **45%**
- 网页性能提高了 **60%**
