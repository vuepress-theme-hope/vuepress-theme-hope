## Plugin Options

### config

- Type: `Record<string, string> | ((app: App) => Record<string, string>)`
- Required: Yes

Redirect map.

### hostname

- Type: `string`
- Required: No

Domain name to redirect to.

## Frontmatter options

### redirect

- Type: `string`
- Required: No

The link which this page redirects to.
