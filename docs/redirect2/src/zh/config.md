## 插件选项

### config

- 类型: `Record<string, string> | ((app: App) => Record<string, string>)`
- 必填: 是

重定向映射。

### hostname

- 类型: `string`
- 必填: 否

重定向到的域名。

## Frontmatter 选项

### redirectFrom

- 类型: `string | string[]`
- 必填: 否

重定向到该页面的地址。

### redirectTo

- 类型: `string`
- 必填: 否

该页面重定向到的地址。
