## How to use

### Install

::: code-tabs#shell

@tab pnpm

```bash
pnpm add -D vuepress-plugin-lightgallery@next
```

@tab yarn

```bash
yarn add -D vuepress-plugin-lightgallery@next
```

@tab npm

```bash
npm i -D vuepress-plugin-lightgallery@next
```

:::

### Usage

::: code-tabs#language

@tab TS

```ts
// .vuepress/config.ts
import { lightgalleryPlugin } from "vuepress-plugin-lightgallery";

export default {
  plugins: [
    lightgalleryPlugin({
      // your options
    }),
  ],
};
```

@tab JS

```js
// .vuepress/config.js
import { lightgalleryPlugin } from "vuepress-plugin-lightgallery";

export default {
  plugins: [
    lightgalleryPlugin({
      // your options
    }),
  ],
};
```
