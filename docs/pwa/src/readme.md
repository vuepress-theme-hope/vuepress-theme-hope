This plugin will turn on Progressive Web App Support.

## How to use

### Install

```bash
npm i -D @mr-hope/vuepress-plugin-pwa
```

Or

```bash
yarn add -D @mr-hope/vuepress-plugin-pwa
```

### Usage

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "@mr-hope/pwa",
      {
        // your options
      },
    ],
  ],
};
```
