---
title: AudioPlayer
---

在 Markdown 文件中嵌入音频。

<!-- more -->

## 示例

一个音频播放器:

<AudioPlayer src="/sample.mp3" />

```md
<AudioPlayer src="/sample.mp3" />
```

一个拥有标题和封面的音频播放器:

<AudioPlayer
  src="/sample.mp3"
  title="A Sample Audio"
  poster="/logo.svg"
/>

```md
<AudioPlayer
  src="/sample.mp3"
  title="A Sample Audio"
  poster="/logo.svg"
/>
```

## 属性

### src

- 类型: `string`
- 必填: 是

音频文件地址

### width

- 类型: `string | number`
- 默认值: `100%`

音频组件宽度。

### type

- 类型: `string`
- 必填: 否

音频类型。

::: note

如果你的服务器不能为音频文件返回正确的 mime 类型，你应该指定它。 (例如：`audio/mp3`)

:::

### title

- 类型: `string`
- 必填: 否

音频标题

### poster

- 类型: `string`
- 必填: 否

音频封面
