## `auto`

> 基于主题模式

@slidestart

## 幻灯片标题

一个拥有文字和 [链接](https://mrhope.site) 的段落

---

## 代码高亮

```js [2-4|1-5]
const add = (a, b) => {
  if (typeof b === "undefined") return a + 1;

  return a + b;
};
```
