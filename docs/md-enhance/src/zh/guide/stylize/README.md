该插件提供以下功能来样式化您的内容：

- 警告: GFM 风格的警告
- 自定义对齐: 自定义内容对齐方式
- 提示容器: 将内容包裹在不同类型的提示框中
- 属性支持: 为 Markdown 内容添加属性
- 标记: 突出显示内容
- Stylize: 可定制的令牌样式器

<!-- #endregion intro -->
