## type

- 类型: `'valine' | 'vssue' | 'disable'`
- 必填: 是

使用 Valine 还是 Vssue。设置为 `'disabled'` 仅会禁用评论功能。

## comment

- 类型: `boolean`
- 默认: `true`

是否默认启用评论功能。
