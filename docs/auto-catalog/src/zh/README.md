## 使用插件

### 安装

::: code-tabs#shell

@tab pnpm

```bash
pnpm add -D vuepress-plugin-auto-catalog
```

@tab yarn

```bash
yarn add -D vuepress-plugin-auto-catalog
```

@tab npm

```bash
npm i -D vuepress-plugin-auto-catalog
```

:::

### 使用

::: code-tabs#language

@tab TS

```ts
// .vuepress/config.ts
import { autoCatalogPlugin } from "vuepress-plugin-auto-catalog";

export default {
  plugins: [
    autoCatalogPlugin({
      //插件选项
    }),
  ],
};
```

@tab JS

```js
// .vuepress/config.js
import { autoCatalogPlugin } from "vuepress-plugin-auto-catalog";

export default {
  plugins: [
    autoCatalogPlugin({
      //插件选项
    }),
  ],
};
```

:::
