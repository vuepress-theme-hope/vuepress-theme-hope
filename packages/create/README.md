Create template shortcut for vuepress-theme-hope.

## Usage

```bash
# npm
npm init vuepress-theme-hope <dir>

# yarn
yarn create vuepress-theme-hope <dir>

# pnpm
pnpm create vuepress-theme-hope <dir>
```
