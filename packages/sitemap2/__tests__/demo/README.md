## 使用插件

### 安装

```bash
npm i -D vuepress-plugin-sitemap2
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "sitemap2",
      {
        // 你的配置选项
      },
    ],
  ],
};
```
