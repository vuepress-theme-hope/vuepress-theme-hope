Copyright plugin for VuePress.

## Usage

```bash
# pnpm
pnpm add -D vuepress-plugin-copyright2@next
# npm
npm i -D vuepress-plugin-copyright2@next
# yarn
yarn add -D vuepress-plugin-copyright2@next
```
