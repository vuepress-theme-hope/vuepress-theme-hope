## Demo

Install VuePress Theme Hope:

::: code-tabs#shell

@tab pnpm <Badge text="Recommended" />

```bash
pnpm add -D vuepress-theme-hope
```

@tab yarn

```bash
yarn add -D vuepress-theme-hope
```

@tab:active npm

```bash
npm i -D vuepress-theme-hope
```

:::
